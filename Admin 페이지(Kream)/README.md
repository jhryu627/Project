# Admin Page Project

## 📋 개요

관리자 페이지 및 회원가입 폼 UI가 포함된 프로젝트입니다.  
Bootstrap 5 기반으로 반응형 레이아웃과 다크모드, 모달 UI를 적용하였습니다.

---

## ✨ 주요 기능

- 🕐 현재 시간 표시
- 🌗 다크 모드 전환
- 🧾 상품 테이블 조회
- 📝 회원가입 버튼 (좌측 상단 고정)
- ✅ 회원가입 모달 폼
